# HubSyncBr 0.4 — Window Manager / Hub View

Objetivo: transformar o núcleo do app em um hub de janelas web, não apenas duas telas fixas.

Implementado nesta versão:
- Até 8 janelas abertas.
- Até 4 janelas visíveis ao mesmo tempo.
- Gerenciador visual de janelas.
- Janela minimizada/oculta sem precisar fechar tudo.
- Botão + Janela.
- Layout igual, primeira maior, segunda maior e grade 2x2.
- Popup visual escuro combinando com o app.
- Botões inferiores compactados: Foco, Som, Desk, Ext.
- Base técnica para grupos estilo Chrome em versões futuras.

Próximos passos planejados:
- Grupos de janelas: Futebol, Lives, Trabalho etc.
- Salvar/restaurar grupos.
- Reordenar janelas por arrastar.
- Divisor arrastável ou controle por barra.
- Premium futuro para 4+ janelas simultâneas em aparelhos fortes.
