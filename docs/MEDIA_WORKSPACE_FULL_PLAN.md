# HubSyncBr 0.7.3 — Media Workspace Separado

Mudanças:
- Media Hub agora tem núcleo próprio.
- Home/Browser ficam separados do Media Hub.
- Ao entrar em Media Hub, aparece um workspace vazio se ainda não houver mídia aberta.
- O botão + cria janela de mídia quando o usuário está no Media Hub.
- Janelas Media Hub ocultam a barra de URL.
- Imagem, GIF e vídeo local ocupam o máximo da janela.

Regra de uso:
- Home = janelas web/navegação/transmissões online.
- Media Hub = janelas de mídia local/offline.
