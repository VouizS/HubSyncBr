# HubSyncBr 0.4.1 — Clean Core / Workspace

Objetivo: aumentar a área útil do núcleo para 4 janelas e preparar a interface para o Hub de Janelas.

Implementado nesta versão:
- Topbar com ícones limpos, sem cards/bordas pesadas.
- Tooltips por toque longo nos botões principais.
- Sidebar mantida com cards, pois é navegação principal.
- Em modo 3/4 janelas, a sidebar recolhe automaticamente para liberar espaço.
- Controles Foco/Som/Desk/Ext saem do rodapé fixo da janela.
- Cada janela passa a ter um menu próprio de ações.
- Barra de URL mais compacta.
- Espaçamentos menores entre janelas.
- Núcleo mais amplo para Grid View.

Próximos passos planejados:
- Slider/escala do workspace.
- Minimizar janela em pílula/card inferior.
- Grupos de janelas estilo Chrome.
- Arrastar janelas para reorganizar e agrupar.
