# Referência visual HubSyncBr

A interface da Sprint 0.1 foi planejada para seguir a imagem de referência enviada pelo usuário:

- visual premium escuro;
- sidebar lateral;
- modo duas telas;
- topbar com troca de telas;
- cards arredondados;
- acentos roxo e azul;
- foco em uso horizontal/rotação;
- identidade de hub multitela.

A imagem de referência não foi incluída no APK para evitar uso indevido de capturas, marcas, transmissões ou conteúdo de terceiros.
